# Vault take home interview task

## Pre-requisites

You must have Node.js v22.14.0 or `nvm` (node version manager) installed. Tests do not pass in node v20 and below due to using the latest version of MSW (Mock Service Worker).

```bash
nvm install 22.14.0
```

## Running the app

```bash
npm install
npm run dev
```

## Running the tests

```bash
npm run test
```
